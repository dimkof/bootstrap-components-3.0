Radio

Example

<span data-toggle="radio" class="radio btn btn-default"></span>
<span data-toggle="radio" class="radio btn btn-default" disabled></span>

Options

Options should be passed via data attributes. Append the option name to data-, as in data-to="".

Name Type Default Description
to element null Set target element value when checkbox clicked. (Suggest setting to <input> element)
value string null Set radio button value.

Methods

.radio(option)

Set radio check status.

$('#myRadio').radio({
    checked: true
})

Events

Event Type Description
change Trigger when checkbox clicked. It will also trigger change event for data-to element. 
$('#myRadio').change(function () {
    // do something...
})

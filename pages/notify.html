Notify

Notify helps creating float alert box.

Usage

Via JavaScript

Create notify through $.notify function.

$.notify({
    title: 'Hi',
    content: 'This is a notify!',
});

Options

Options need be passed via JavaScript.

Name Type Default Description
title string Notification Set the notify box title.
content string | html null Set the notify box content.
position string right,top Set notify position: left, right, top, bottom. You can mix then up as left,top (it's same as top,left)
type string null Set notify box style: normal, warning, info, danger, success
timeout number 4000 Set notify box hide time delay. -1 make it never auto hide.
overtimeout number 1000 Set notify box hide time delay after mouse move out.
queuetimeout number 1000 Set notify box hiden speed interval between 2 notification in same the region.
region string '' Set region of notify box. Saming region notify box will not keep out other one in same region.

Events

Event Type Description
callback Trigger when notification created. Use $(this) to get created element. 
$.notify({
    title: 'Hi Notify!',
}, function() {
    // do something...
});

Slider

Slider is a useful component which display number as a slider.

You can use it through html defination.

<div class="progress">
    <button type="button" class="btn btn-primary slider" data-toggle="slider"></button>
</div>
<div class="progress">
    <button type="button" class="btn btn-primary slider" data-toggle="slider"></button>
    <button type="button" class="btn btn-primary slider" data-toggle="slider"></button>
</div>

But if you want to get more powerful slider support. Please call function to initial it.

$('#myElement').slider();

$('#myElement').slider({value: [0, 25, 50, 75]});

Usage

Slider container <div> use class of .progress

Via data attributes

For slider <button> use class of .btn and .slider, so you can using buton class for the slider style. Addtinally, to trigger the slider movable, you need set data-toggle="slider" on the slider button.

<button class="btn btn-primary slider" data-toggle="slider"></button>

Via JavaScript

Call a modal with id sliderContainer with a single line of JavaScript:

$('#sliderContainer').slider(options)

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-to="".

Name Type Default Description
min number 0 Attr of Slider Container. Set minimum value for the slider.
max number 100 Attr of Slider Container. Set maximum value for the slider.
to element null Attr of Slider Button. Set target element value when slider button moves. (Suggest setting to <input> element)
number number 1 Set number of slider button displayed in the slider container.
value array of number null Set value for each slider button. If not set the number of slider buttons, slider will auto generate correct number of slider buttons when you give the values.
single boolean false Move slider button will not influence others if true.
mixed boolean false

Slider button can move every where without order if true.

PS: If set mixed as true, single will always be true.

Methods

.slider(option)

Activates your content as a slider. Accepts an optional options object.

$('#mySliderContainer').slider({
    min: 0,
    max: 100,
    value: [5, 9, 11, 25, 38],
    single: true
})

Events

Event Type Description
change Trigger on each slider button when user move it. It will also trigger change event for data-to element. 
$('#mySliderButton').change(function () {
    // do something...
})

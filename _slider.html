Slider

Slider is a useful component which display number as a slider.

You can use it through html defination.

<div class="progress">
    <button type="button" class="btn btn-primary slider" data-toggle="slider"></button>
</div>
<div class="progress">
    <button type="button" class="btn btn-primary slider" data-toggle="slider"></button>
    <button type="button" class="btn btn-primary slider" data-toggle="slider"></button>
</div>

But if you want to get more powerful slider support. Please call function to initial it.

$(element).slider();

$(element).slider({value: [0, 25, 50, 75]});

Usage

Slider container <div> use class of .progress

Via data attributes

For slider <button> use class of .btn and .slider, so you can using buton class for the slider style. Addtinally, to trigger the slider movable, you need set data-toggle="slider" on the slider button.

<button class="btn btn-primary slider" data-toggle="slider"></button>

Getting Start

Bootstrap Components is component lib which used for Bootstrap with the same theme.

This page is about Bootstrap Components works for Bootstrap v3. So if you want to get Bootstrap Components works for Bootstrap v2 please click here.

Download jQuery and Bootstrap

The first step of using Bootstrap Components needs jQuery and Bootstrap include.

Download jQuery Download Bootstrap

Download Bootstrap Components

Download as .zip Download as .tar.gz or Go to Github

What's included

bootstrap-components/
├── css/
│   └── bootstrap-components.min.css
└── js/
    └── bootstrap-components.min.js

How to use it?

Include bootstrap-components.min.js and bootstrap-components.min.css into html document.

inclued jQuery and Bootstrap first

Bootstrap Compoents requires fully function jQuery and Bootstrap support. Please include them firstly to make Bootstrap Components work.

<script src="bootstrap-components/js/bootstrap-components.min.js"></script>
<link href="bootstrap-components/css/bootstrap-components.min.css" rel="stylesheet" media="screen">
